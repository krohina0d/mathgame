# Чат с ИИ - Инструкция по настройке

## Настройка админа

Чтобы сделать пользователя админом, нужно вручную изменить роль в базе данных Firebase:

1. **Откройте Firebase Console**: https://console.firebase.google.com/
2. **Выберите ваш проект** (mathgame-530a3)
3. **Перейдите в Firestore Database**
4. **Найдите коллекцию `users`**
5. **Найдите документ с вашим UID** (ID пользователя)
6. **Измените поле `role` с `"user"` на `"admin"`**

## Функции админа

- ✅ **Управление промптами**: Создание и редактирование системных промптов
- ✅ **Активация промптов**: Выбор активного промпта для всех пользователей
- ✅ **Доступ к админ-панели**: Кнопка "Админ" в интерфейсе чата

## Функции пользователей

- ✅ **Вход через Google**: Быстрая авторизация
- ✅ **Регистрация по email**: Создание аккаунта с паролем
- ✅ **Чат с ИИ**: Общение с использованием активного промпта
- ✅ **История сообщений**: Сохранение диалога в течение сессии

## Настройка API ключа

1. Создайте файл `.env` в корне проекта
2. Добавьте ваш OpenAI API ключ:
```
VITE_OPENAI_API_KEY=your_openai_api_key_here
```

## Технологии

- **Frontend**: React + TypeScript + Material UI
- **Backend**: Firebase (Authentication + Firestore)
- **AI**: OpenAI GPT-4-turbo
- **Сборка**: Vite
