# Чат с ИИ

Веб-приложение для общения с ИИ на основе GPT с системой аутентификации и админ-панелью.

## Возможности

### Для пользователей
- 🔐 **Вход через Google** или регистрация по email
- 💬 **Чат с ИИ** с использованием GPT-4-turbo
- 📝 **История сообщений** в течение сессии
- 🎨 **Современный интерфейс** на Material UI

### Для администраторов
- ⚙️ **Админ-панель** для управления системными промптами
- 📋 **Создание и редактирование промптов**
- 🔄 **Активация/деактивация промптов**
- 👥 **Система ролей** (админ/пользователь)

## Настройка

1. **Установите зависимости:**
```bash
npm install
```

2. **Создайте файл `.env` в корне проекта:**
```
VITE_OPENAI_API_KEY=your_openai_api_key_here
```

3. **Получите API ключ на [OpenAI Platform](https://platform.openai.com/api-keys)**

4. **Настройте админа** (см. [SETUP.md](./SETUP.md))

5. **Запустите приложение:**
```bash
npm run dev
```

## Технологии

- **Frontend**: React + TypeScript + Material UI
- **Backend**: Firebase (Authentication + Firestore)
- **AI**: OpenAI GPT-4-turbo
- **Сборка**: Vite
